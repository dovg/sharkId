# Юзкейзы

---

## UC-01 Авторизация

Цель: ограничить доступ к системе.

Сценарий:

1. Пользователь открывает приложение.
    
2. Вводит логин/пароль.
    
3. Получает токен/сессию.
    
4. Получает доступ к функционалу.
    

---

## UC-02 Создание погружения

Цель: создать DiveSession.

Сценарий:

1. Пользователь нажимает “Новое погружение”.
    
2. Выбирает локацию из справочника.
    
3. Указывает дату/время начала и окончания.
    
4. Добавляет комментарий.
    
5. Сохраняет.
    

---

## UC-03 Управление локациями

Сценарий добавления:

1. Пользователь открывает раздел “Локации”.
    
2. Нажимает “Добавить”.
    
3. Заполняет страну, спот, координаты.
    
4. Сохраняет.
    

---

## UC-04 Загрузка фото

Сценарий:

1. Пользователь открывает погружение.
    
2. Загружает пачку фото.
    
3. Фото сохраняются в MinIO.
    
4. Создаются записи Photo.
    
5. Автоматически запускается классификация.
    

---

## UC-05 Автоматическая автоидентификация

Сценарий:

1. Фото загружено.
    
2. Система выделяет область морды.
    
3. Генерирует признаки.
    
4. Выполняет поиск ближайших соседей.
    
5. Формирует основной кандидат + топ-5.
    
6. Применяет глобальный порог.
    
7. Фото получает статус ready_for_validation.
    

---

## UC-06 Валидация привязки

Сценарий:

1. Пользователь открывает очередь валидации.
    
2. Видит лучший кандидат + топ-5.
    
3. Выбирает:
    
    - подтвердить кандидата,
        
    - выбрать другую акулу,
        
    - создать новую,
        
    - оставить без привязки.
        
4. Система фиксирует решение.
    

---

## UC-07 Создание новой акулы

Сценарий:

1. Пользователь выбирает “Новая акула”.
    
2. Система предлагает временное имя.
    
3. Пользователь подтверждает или меняет имя.
    
4. Создаётся запись Shark.
    
5. Фото привязывается.
    

---

## UC-08 Подтверждение наблюдения

Сценарий:

1. Система извлекает метаданные.
    
2. Создаёт черновик Observation.
    
3. Пользователь редактирует/подтверждает.
    
4. Наблюдение сохраняется как confirmed.
    

---

## UC-09 Каталог акул

Сценарий:

1. Пользователь открывает каталог.
    
2. Ищет по имени.
    
3. Открывает карточку.
    
4. Видит фото и историю встреч.
