# Зафиксированные решения

- Реализована простая ролевая модель: viewer / editor / admin.
- Авторизация обязательна (JWT).
- Справочник локаций редактируемый.
- Порог уверенности для идентификации глобальный (не на уровне акулы).
- Топ кандидатов = 5.
- Форматы изображений: JPEG, PNG (макс. 50 МБ).
- Форматы видео: MP4, MOV, AVI, MKV, WebM (макс. 500 МБ).
- Экспорт данных не требуется.
- Бэкап обязателен (БД + MinIO).
- /internal/ API удалён; управление пользователями — через JWT-защищённый /users (admin only).
- Прототип (prototype/) удалён после завершения разработки.
