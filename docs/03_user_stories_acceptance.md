# User Stories (MVP)

---

## US-01 Авторизация

- Неавторизованный пользователь не имеет доступа.
    
- Сессия ограничена по времени.
    

---

## US-02 Локации

- Есть поиск по стране/споту.
    
- Можно добавить новую запись.
    
- Новая запись доступна сразу.
    

---

## US-03 Погружения

- Можно создать DiveSession.
    
- Фото привязываются к погружению.
    

---

## US-04 Автоидентификация

- Запускается автоматически.
    
- Показывается лучший кандидат + топ-5.
    
- Используется глобальный порог.
    
- Требуется подтверждение.
    

---

## US-05 Новая акула

- Предлагается временное имя.
    
- Можно изменить вручную.
    
- Имя может оставаться temporary.
    

---

## US-06 Наблюдение

- Метаданные сохраняются полностью.
    
- Пользователь подтверждает всегда.
    
- Можно заполнить недостающие поля вручную.
    

---

## US-07 Хранение фото

- Фото хранятся в MinIO.
    
- Поддерживаются jpeg/png.
    

---

## US-08 Бэкап

- Регулярный бэкап БД.
    
- Регулярный бэкап MinIO.
    
- Есть процедура восстановления.
