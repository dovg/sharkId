# Архитектура системы

## Стек
Backend: Python (FastAPI)
Frontend: React
Хранилище файлов: MinIO
БД: PostgreSQL
ML-модуль: Python (отдельный сервис или модуль)

## Логическая схема

React SPA
    ↓
FastAPI Backend
    ↓
PostgreSQL
MinIO
ML Service

## Основные компоненты

### Backend (FastAPI)
- REST API
- Авторизация (JWT)
- Управление сущностями
- Работа с MinIO
- Оркестрация ML-классификации

### ML-модуль
- Выделение области морды
- Генерация эмбеддингов
- Поиск ближайших соседей
- Возврат топ-5 кандидатов

### Frontend (React)
- Авторизация
- Каталог акул
- Погружения
- Очередь валидации
- Управление локациями

## Бэкап
- Регулярный дамп PostgreSQL
- Регулярный snapshot MinIO
